Learn How It Works

A quick guide to CNN-based signature verification and Grad-CAM explainability.

CNN for Signature Features

The convolutional network learns stroke flow, pressure-like texture patterns, curvature behavior, and spatial consistency from labeled signature images.

Grad-CAM Visualization

Grad-CAM highlights image regions that contributed most to the model's decision, making the result interpretable instead of a black-box score.

Decision Output

The model returns a confidence value and a class prediction (genuine or forged). Higher confidence indicates a stronger pattern match with learned behavior.
